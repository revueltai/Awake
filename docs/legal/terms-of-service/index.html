Awakeme Terms of Service & Privacy Policy

This privacy policy governs the use of the software application Awakeme ("Application") for mobile devices. The Application is aimed to help a person ("user") reach a destination in the map based on the persons's location in the map, and a given destination. The Application is distributed as Open Source under the MIT License, is free, and the code that composes it can be found here

What information does the Application obtain and how is it used?

The Application obtains information about the Geolocation of the user and this used internally to locate the current position of the person in the map.

User Provided Information

The Application requests the user to enable Geolocation and share this information. The Application requests this when downloaded and being used for the first time. If the user does not allow Geolocation then the Application will not be able to function properly.

Automatically Collected Information

The Application may collect certain information automatically, including, but not limited to, the type of mobile device of the user, the user's mobile devices unique device ID (UID), the IP address of the user's mobile device, the user's mobile operating system, the type of mobile Internet browsers of the user, and information about the way the Application is used.

Does the Application collect precise real time location information of the device?

Yes. The Application may use GPS technology (or other similar technology) to determine the user's current location. The user's current location will not be shared with other users or partners. If the user does not want the Application to use Geolocation for the purposes set forth above, him/her should turn off the location services for the mobile application located in the account settings or in the user's mobile phone settings and/or within the mobile application. However, this will affect the functioning of the Application.

Do third parties see and/or have access to information obtained by the Application?

No. The Application does not share the information with third parties.

However, we may disclose User Provided and Automatically Collected Information:

  • as required by law, such as to comply with a subpoena, or similar legal process;

  • when we believe in good faith that disclosure is necessary to protect the users rights, protect the user's safety or the safety of others, investigate fraud, or respond to a government request;

Use of other plugins

The Application makes use of other plugins (Google Maps, Google Places). Each plugin is the responsibility of its provider. We have no control over the amount of information that the respective provider collects with the help of its own plugin, or how it then uses that information.

If the user is not yet a customer of the provider or has not (yet) consented there to having its data collected and used, we recommend not clicking on these plugins.

If, on the other hand, the user is already a customer of the plugin provider, it should refer directly to the relevant provider for information on the kind and amount of personal data it collects and uses, and for what purpose.

User opt-out rights

The user can stop all collection of information by the Application by uninstalling the Application. The user may use the standard uninstall processes as may be available as part of the mobile device or via the mobile application marketplace or network.

Data Retention Policy, Managing the user's Information

The Application only uses information about the user's location. It does so temporary while the Application is active. No data or personal information is retained.

Security

We provide physical, electronic, and procedural safeguards to protect information we process and maintain. We limit access to this information to authorized employees and contractors who need to know that information in order to operate, develop or improve our Application. Please be aware that, although we endeavor provide reasonable security for information we process and maintain, no security system can prevent all potential security breaches.

Changes

This Privacy Policy may be updated from time to time for any reason. Any changes to our Privacy Policy will be notified by posting on this page. You are advised to consult this Privacy Policy regularly for any changes, as continued use is deemed approval of all changes.

Your Consent
